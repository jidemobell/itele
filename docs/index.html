This is a job application API. you can use this API to get users activities and retireve information about a user.

List Users

List top users by activaty within the week and latest applied listings

Property Description
page a page number
GET
/topActiveUsers?page={pageNumber}

Example request

http://localhost:4000/topActiveUsers?page={pageNumber}

Get User

Takes user id and returns: user info; connected companies; listings created by the user; applications the user has made with the info of the listing the application is made to.

Property Description
id a user id
GET
/users?id={user.id}

Example request

http://localhost:4000/users?id={user.id}
